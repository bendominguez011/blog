In a previous tutorial we set up a very basic skeleton to start our flask app. In this post, we are going to set up the database. First, we have to install the appropriate modules and packages. In the terminal run the command:

flask/bin/pip install flask-sqlalchemy

This is our one module we are going to use to help us write database models. Flask SQL-ALchemy is a helpre tool for SQL and Python, and has an included Object-Relational Mapper. All this means is that we can create dataase tables and query using Python instead of SQL (Standard Querying Language), and this is done using Object Oriented Programming. For now, we are going to need to find a URI, or path, to our current directory and store it as a string in the variable SQLALCHEMY_DATABASE_URI So in the same module app.py,


from flask import Flask
from flask.ext.sqlalchemy import SQLAlchemy
import os

app = Flask(__name__)



base_directory = os.path.dirname(__file__)
SQLALCHEMY_DATABASE_URI = 'sqlite:///' + os.path.join(os.path.abspath(base_directory), 'app.db')

db = SQLAlchemy(app)



app.route('/')

      return "Hello World"

if __name__ == "__main__":
      app.run(debug=True, host="0.0.0.0")
